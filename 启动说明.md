# 阿里云百炼文生图 Web UI 快速启动

## 启动方式

### 方式一：双击启动（推荐）
直接双击 `启动WebUI.bat` 文件

### 方式二：Python启动器
```bash
python 启动器.py
```

### 方式三：直接运行
```bash
python bailian_webui.py
```

## 启动后

- 自动在浏览器中打开界面
- 默认访问地址: http://127.0.0.1:7860
- 按 `Ctrl+C` 停止服务

## 常见问题

**Q: 双击bat文件闪退**
A: 请确保已安装 Python 并添加到环境变量

**Q: 提示端口被占用**
A: 脚本会自动尝试其他端口（7861, 7862, 7870, 8000, 8080）

**Q: 浏览器没有自动打开**
A: 手动访问 http://127.0.0.1:7860
